PORTFOLIO IMAGES
================

Drop 6 picnic photos in this folder named:
  1.jpg  2.jpg  3.jpg  4.jpg  5.jpg  6.jpg

Recommended sizes (matches the homepage mosaic):
  1.jpg  — TALL  cell    ~600 × 880 px (portrait)
  2.jpg  — square cell   ~600 × 440 px
  3.jpg  — square cell   ~600 × 440 px
  4.jpg  — square cell   ~600 × 440 px
  5.jpg  — square cell   ~600 × 440 px
  6.jpg  — WIDE cell    ~1200 × 440 px (landscape)

Keep file sizes under ~300 KB each for fast loading.
Save as JPG (or WebP) — not PNG — for photos.

To swap a photo: replace the file with the same filename.
To change a caption: edit the Portfolio tab of the Google Sheet.
To use a different image URL (e.g. one hosted elsewhere): paste the full URL into the Sheet's image_url column.
